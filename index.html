How I use Travis CI to automatically deploy this blog to GitHub Pages

I host this blog for free with GitHub Pages. Pages integrates with Jekyll, a popular static site generator. When you make changes to your Jekyll site, GitHub automatically generates the public HTML files from the source code in your repository. Simply git push and, et voilà, your site is live! Jekyll is great, but I prefer Hugo.
